# DebugIt

> Send us the mess. Get back a working app.

DebugIt is a human-led software rescue shop for broken, unfinished, unstable, and deployment-blocked applications.

## The promise

Your code remains your code. Your idea remains your idea. We are here to repair it, not take it.

## Initial product surfaces

- Landing page
- Code Ownership Treaty
- Secure project intake
- Customer project-status portal
- Admin debugging queue
